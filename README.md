# CatLaser
Simple Arduino code to keep your cat entertained.

Here's what you need:
- Arduino Uno
- Wires
- 2 Servos
- Something to mount the servos on (vertically and horizontally)
- Laser pointer

The laser moves in a configurable area defined by points.
